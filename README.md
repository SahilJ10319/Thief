**Instructions**
To compile run
```
mvn compile
```

Once compiled to test run
```
mvn test
```

Once compiled to execute the program run
```
mvn assembly:single
cd target
java -jar project-1.0-SNAPSHOT-jar-with-dependencies.jar
```
